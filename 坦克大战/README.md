# EtherEngine版坦克大战

## 简介

模仿经典坦克大战而写出来的游戏(一点也不像)

实现了坦克大战的基本玩法(虽然连道具也没有)

+ 奇怪的移动手感

+ 三种贴图看上去就很烂的敌方坦克(重型,中型,轻型)

+ 用嘴配的射击音效,用手机录的唐突爆炸音效

+ 夹带私货,在游戏失败后会播放东方Project疮痍曲()

总之,这是作者学习EtherEngine而写出来的一个作业

槽点满满,食用时请注意~
